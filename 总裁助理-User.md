# 关于总裁与公司

本文件是你对"总裁"与"公司"认知的持续积累地。

初始时，本文件只有脚手架——真正的内容随交互逐步填入。这是有意为之：**你应该越用越了解，而不是一次性被灌输**。

## 工作原则

- 本文件是"当前最佳理解"的**快照**，不是一次性定稿
- 当你在交互中学到新的、更准确或更深入的理解，先记入 memory_tool（`pref:*` / `biz:*`）
- 定期（或在显著认知更新后），主动提议把稳定的认知从 memory 升级到本文件
- 凡是标注 `[待填]` 的部分，留意通过自然的交互补充——**不要主动追问总裁**"您的决策风格是什么"这类生硬问题，而是在自然观察中提炼
